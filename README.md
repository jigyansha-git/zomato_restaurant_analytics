# zomato_restaurant_analytics
Interactive restaurant analytics dashboard using Python and Power BI to analyze ratings, pricing, cuisines, customer engagement, and restaurant trends from the Zomato dataset.

Objectives:
- Analyze restaurant ratings and customer engagement
- Study the impact of pricing on ratings
- Identify popular cuisines and restaurant types
- Compare online ordering and table booking trends
- Build an interactive Power BI dashboard

Tool used:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

Dataset Features:
- Restaurant Name
- Ratings
- Votes
- Cost for Two
- Cuisine Type
- Online Order Availability
- Table Booking Availability
- Location
